Apresentação da API

Esta API foi desenvolvida utilizando o framework Flask para Python. Ela foi criada com o propósito de fornecer funcionalidades simples, como retornar saudações personalizadas e realizar operações matemáticas básicas como soma. A API foi projetada para ser fácil de usar e para facilitar o desenvolvimento de sistemas que precisem de interações simples com o backend.

A API oferece três endpoints principais:

  • Home: A página inicial da API.
  • Saudação: Retorna uma saudação personalizada.
  • Soma: Realiza a soma de dois números fornecidos via JSON.

1. End Point: Home

URL: /

Método: GET

Descrição: Retorna a página inicial.

2. End Point: Saudação

URL: /saudacao?nome={nome}

Método: GET

Descrição: Retorna uma saudação personalizada com o nome enviado como parâmetro na query string.

Exemplo de chamada: /saudacao?nome=Andrezinho

Resposta: "Olá, {nome} seja muito bem-vindo(a) a minha API Flask"

3. End Point: Soma

URL: /soma

Método: POST

Descrição: Recebe um objeto JSON no corpo da requisição e retorna a soma dos dois números fornecidos.

Exemplo de JSON enviado:
        {
            "operando1": 10,
            "operando2": 5
        }

Resposta: "A soma de {operando1} + {operando2} é {soma}"
